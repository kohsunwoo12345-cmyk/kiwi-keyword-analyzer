MCP 연결 · Claude에서 바로 생성

Claude(데스크톱·웹·Cursor·Claude Code)에 연결하면 대화창에서 바로 영상·이미지 생성을 시킬 수 있습니다. 생성 1건마다 내 계정 크레딧에서 차감됩니다.

개발자 문서 열기 · MCP 연동 방식 자세히 보기 ↗

내 전용 연결 주소 (개인 토큰 포함)

이 주소에는 내 개인 토큰이 들어 있어 남에게 공유 금지입니다. 유출됐다면 토큰 재발급(기존 연결은 끊깁니다).
generate_imagegenerate_video check_video_statuslist_models

① Claude 데스크톱 · claude.ai (커스텀 커넥터)

설정(Settings) → 커넥터(Connectors) → 커스텀 커넥터 추가(Add custom connector) → 위 서버 주소를 붙여넣고 저장 → 대화에서 “영상 만들어줘”라고 하면 도구가 실행됩니다.
※ Pro/Team/Enterprise 플랜에서 커스텀 커넥터가 지원됩니다.

② Cursor

Settings → MCP → Add new MCP server → Type을 HTTP로 두고 URL에 위 주소를 입력.

③ Claude Code (터미널)

claude mcp add --transport http bygency <URL>
추가 후 /mcp 명령으로 연결 상태를 확인하세요.

④ Claude API (개발자)

"mcp_servers": [{ "type": "url", "url": "<URL>", "name": "bygency" }]
Messages API 요청에 mcp_servers 배열로 추가하고 anthropic-beta: mcp-client-2025-04-04 헤더를 넣으세요.

인증 토큰 (선택 · 권장)

서버 환경변수 MCP_AUTH_TOKEN을 설정하면 아무나 접근하지 못합니다. 그때는 주소 뒤에 /토큰을 붙이거나(<URL>/토큰), 헤더에 Authorization: Bearer 토큰을 추가하세요.
연결 확인: 브라우저로 서버 주소를 열었을 때 status: ok JSON이 보이면 정상입니다.

AI 모델 API 연결
제공사별 API 키를 입력하면 해당 모델은 실제 생성으로 동작합니다. 키가 없는 모델은 데모 영상으로 대체됩니다. 키는 이 브라우저에만 저장됩니다.
키는 서버로 전송되지 않고 localStorage 에만 저장됩니다.
저장된 워크플로
이 브라우저에 저장된 워크플로 목록입니다. 불러오면 현재 작업이 대체됩니다.
생성 비용
내 계정에서 실행한 실제 생성의 실단가 기준 비용입니다(영상=초당×길이×해상도, 이미지=장당). 공개 단가 기준 추정치이며, 정확한 청구액은 제공사 콘솔에서 확인하세요.

claude mcp add --transport http bygency /api/mcp
